# 📱 APRO App

Aplikasi **APRO** adalah aplikasi berbasis Flutter yang dikembangkan dari sistem web menjadi aplikasi Android untuk mempermudah penggunaan secara mobile.

---

## 🚀 Download Aplikasi

Silakan download file APK melalui link berikut:

🔗 https://drive.google.com/file/d/1Cu_ZYraNX8PAUFAn2G38DOhS-J-NU-yg/view?usp=sharing

---

## 📦 Cara Install

1. Download file **APRO.apk**
2. Buka file APK
3. Aktifkan **"Install dari sumber tidak dikenal"** jika diminta
4. Install aplikasi hingga selesai
5. Buka aplikasi dan gunakan

---

## ⚙️ Teknologi yang Digunakan

- Flutter
- Dart
- Web-based system integration

---

## 📌 Catatan

- Pastikan perangkat Android mengizinkan instalasi dari sumber luar Play Store
- Jika aplikasi tidak bisa diinstall, coba cek versi Android atau ulangi download
